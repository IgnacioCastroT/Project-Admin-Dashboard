# Admin Dashboard

A responsive admin dashboard built with HTML and CSS as part of [The Odin Project](https://www.theodinproject.com/) Intermediate HTML and CSS course.

## About

This project focuses on practicing **CSS Grid** for layout design, including nested grids for the sidebar, header, and main content areas. The dashboard includes a project cards section, announcements panel, and a trending users section.

## Built With

- HTML5
- CSS3 (Grid, Flexbox)
- [Material Design Icons](https://materialdesignicons.com/)
- [Google Fonts – Roboto](https://fonts.google.com/specimen/Roboto)
